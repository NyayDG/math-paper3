Integration of Irrational Algebraic Functions

Integration of irrational algebraic functions involves finding the
antiderivative of functions that cannot be expressed as a ratio of two polynomials,
such as functions involving radicals or square roots.

One common technique for integrating these types of functions is called substitution.
The idea behind substitution is to make a change of variable that simplifies the integral,
and then find the antiderivative of the resulting expression.

For example, consider the integral:
∫ (x^2 + 1)^(3/2) dx

We can use the substitution u = x^2 + 1, which simplifies the integral to:
∫ u^(3/2) du

Now, we can integrate u^(3/2) using the power rule for integration:
∫ u^(3/2) du = (2/5)u^(5/2) + C

Substituting back u = x^2 + 1, we get:
∫ (x^2 + 1)^(3/2) dx = (2/5)(x^2 + 1)^(5/2) + C

Another technique for integrating irrational algebraic functions is to use
trigonometric substitutions. This involves using trigonometric identities to
convert the integral into a form that can be integrated using trigonometric functions.

For example, consider the integral:
∫ (x^2 + 4)^(3/2) dx

We can use the substitution x = 2tanθ, which simplifies the integral to:
∫ 16sec^4θ dθ

Now, we can use the trigonometric identity sec^2θ = tan^2θ + 1 to rewrite the integrand as:
16(sec^2θ + 1)^2 sec^2θ dθ

Simplifying further using the identity 1 + tan^2θ = sec^2θ, we get:
16(tan^2θ + 2)sec^4θ dθ

Now, we can integrate using the substitution u = tanθ, which gives:
∫ 16(u^2 + 2)(1 + u^2) du

Expanding and integrating term by term, we get:
∫ 16u^4 + 32u^2 + 16 du = (4/5)u^5 + (32/3)u^3 + 16u + C

Substituting back u = tanθ and simplifying using trigonometric identities, we get:
∫ (x^2 + 4)^(3/2) dx = (8/15)(x^2 + 4)^(5/2) - 8x^2 + C

Overall, integration of irrational algebraic functions requires careful
application of substitution, trigonometric identities, and algebraic
manipulations to simplify the integrand and find the antiderivative.

Integration by Rationalisation

Integration by rationalization is a technique used in calculus to simplify
an integrand by multiplying the numerator and denominator of the fraction
by a conjugate factor. The goal of this technique is to create a simpler

    The conjugate is found by changing the sign between the terms in the denominator.

    we multiply both the numerator and denominator of the fraction by √(x+1):

    = ∫ √(x+1)/(x+1) dx

    we can use a substitution to simplify the integral:
    Let u = x+1. Then du/dx = 1, and dx = du. Substituting this into the integral gives:
    ∫ √(x+1)/(x+1) dx = ∫ √u/u du
